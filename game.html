Friends That Game

Distributed scorekeeper for card and board games. Everyone enters their own score — no more passing the phone around.

How it works

01

Host starts a game

Pick from your saved game types or create one on the fly. Set the rules once, reuse them every game night.

02

Everyone joins

Share a 4-letter code or a link. Players join from their own phone — no app required for guests.

03

Each player enters their score

Every player locks in their own score each round. The host sees who's done and can advance when everyone's ready.

04

Live leaderboard

Scores update in real time for everyone. See the Before / This Round / After breakdown each round.

Features

Game type library

Save your group's favorite games — win direction, end conditions, target scores, round limits. Configure once, play forever.

Multi-device scoring

Each player has their own score entry screen. The host has an override view for corrections and force-advancing rounds.

Lock In mechanic

Players confirm their score each round. The host sees lock status at a glance — green, amber, or waiting.

Flexible end conditions

Fixed rounds, first to reach a score, bust out at a threshold, or open-ended (host declares the winner). Works for Phase 10, Mexican Train, and more.

Went Out tracking

For games with a zero-score bonus. Auto-detected when one player hits zero — or host is prompted when it's ambiguous.

Share results

End of game sends a clean text summary via the iOS share sheet. Screenshot-free bragging rights.

Real time

Scores sync instantly across every phone at the table via Firebase.

No app needed

Guests can join via link. Only the host needs the app installed.

Currently in beta

Friends That Game is invite-only while we test with our group. Public launch coming in the future.
